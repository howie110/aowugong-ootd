# Privacy Policy

Last updated: 2026-06-11

每日穿搭 is a local-first Android app for recording outfit photos and metadata.

## Data Collection

The current version does not provide:

- User accounts
- Cloud sync
- Analytics
- Ads
- Crash reporting SDKs
- Remote servers

The app does not intentionally upload outfit records, photos, backup files, or device identifiers.

## Local Data

Outfit records and app settings are stored on the user's device. Photos are selected by the user and managed locally by the app.

Uninstalling the app may remove local app data. Users should export a ZIP backup before uninstalling or migrating devices.

## Android Permissions

The app requests Android media/photo permissions so users can select and manage local outfit images.

## Backups

ZIP backups are created only when the user chooses to export backup data. A backup may include outfit metadata and referenced outfit images. Users are responsible for where they store or share exported backup files.

## Future Changes

If future versions add cloud sync, analytics, crash reporting, ads, or any network-based feature, this policy and the README will be updated before release.
