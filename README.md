# Daily OOTD

本仓库用于开发一款离线优先的 Flutter 移动端应用，目标是帮助用户按天记录每日穿搭，并基于标签回看与筛选历史记录。

当前阶段还未开始编码，先通过文档确认产品范围、技术方案和实施顺序。
