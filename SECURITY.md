# Security Policy

## Supported Versions

Only the latest GitHub Release is actively maintained.

## Reporting a Vulnerability

If you find a security or privacy issue, please do not publish sensitive details in a public issue.

Recommended report path:

1. Open a private contact channel with the maintainer if available on the GitHub profile.
2. If private contact is not available, open a public issue with a minimal title such as `Security report request` and avoid posting exploit details, private data, tokens, or screenshots that contain sensitive information.

Please include:

- A short description of the affected feature
- Steps to reproduce, if safe to share
- Device and Android version
- App version
- Whether local photos, backup ZIP files, or private paths may be affected

The maintainer will review reports on a best-effort basis.
